# 07 - Engenheiro de Prompt

Otimiza prompts manualmente com tecnicas avancadas.

## Informacoes

| Campo | Valor |
|-------|-------|
| **Nome** | Engenheiro de Prompt |
| **Nodes** | 21 |
| **Trigger** | Manual |
| **LLM** | Claude Opus 4 |

## Funcao

1. Recebe prompt atual
2. Analisa estrutura e clareza
3. Aplica tecnicas de prompt engineering
4. Gera versao otimizada
5. Testa com exemplos

## Tecnicas Aplicadas

- Chain of Thought (CoT)
- Few-shot learning
- Role prompting
- Constraint setting
- Output formatting

## Fluxo

```
Prompt Input --> Analise --> Otimizacao --> Teste --> Output
```

## Output

- Prompt otimizado
- Relatorio de mudancas
- Exemplos de teste
