  <files>apps/docs/sql/070_mindflow_schema.sql</files>

Create `apps/docs/sql/070_mindflow_schema.sql` with all 6 tables. Follow the exact schema from RESEARCH.md code examples:

**1. mindflow_boards:**

- id UUID PK, name TEXT NOT NULL DEFAULT 'New Board', description TEXT
- columns JSONB NOT NULL DEFAULT '[]' (array of {id, type, title, settings, width, position})
- column_order TEXT[] NOT NULL DEFAULT '{}'
- icon TEXT DEFAULT 'table-2', color TEXT DEFAULT '#6C5CE7'
- created_by UUID NOT NULL REFERENCES auth.users(id)
- is_archived BOOLEAN NOT NULL DEFAULT false
- created_at TIMESTAMPTZ NOT NULL DEFAULT now(), updated_at TIMESTAMPTZ NOT NULL DEFAULT now()

**2. mindflow_groups:**

- id UUID PK, board_id UUID NOT NULL FK -> mindflow_boards ON DELETE CASCADE
- name TEXT NOT NULL DEFAULT 'New Group', color TEXT DEFAULT '#6C5CE7'
- position TEXT NOT NULL (fractional index)
- is_collapsed BOOLEAN NOT NULL DEFAULT false
- created_at, updated_at

**3. mindflow_items:**

- id UUID PK, board_id UUID NOT NULL FK -> mindflow_boards ON DELETE CASCADE
- group_id UUID NOT NULL FK -> mindflow_groups ON DELETE CASCADE
- parent_id UUID FK -> mindflow_items ON DELETE CASCADE (subitems, Phase 6)
- name TEXT NOT NULL DEFAULT '', column_values JSONB NOT NULL DEFAULT '{}'
- position TEXT NOT NULL (fractional index)
- is_archived BOOLEAN NOT NULL DEFAULT false
- created_by UUID NOT NULL FK -> auth.users(id)
- created_at, updated_at

**4. mindflow_views:**

- id UUID PK, board_id UUID NOT NULL FK -> mindflow_boards ON DELETE CASCADE
- name TEXT NOT NULL DEFAULT 'Main Table'
- type TEXT NOT NULL DEFAULT 'table' CHECK (type IN ('table', 'kanban', 'calendar'))
- position INTEGER NOT NULL DEFAULT 0
- filters JSONB DEFAULT '[]', sort_config JSONB DEFAULT '[]'
- group_by TEXT, hidden_columns TEXT[] DEFAULT '{}'
- column_widths JSONB DEFAULT '{}', kanban_config JSONB, calendar_config JSONB
- is_default BOOLEAN NOT NULL DEFAULT false
- created_by UUID NOT NULL FK -> auth.users(id)
- created_at, updated_at

**5. mindflow_dashboards:**

- id UUID PK, board_id UUID FK -> mindflow_boards ON DELETE SET NULL (null = cross-board)
- name TEXT NOT NULL DEFAULT 'New Dashboard'
- created_by UUID NOT NULL FK -> auth.users(id)
- created_at, updated_at

**6. mindflow_dashboard_widgets:**

- id UUID PK, dashboard_id UUID NOT NULL FK -> mindflow_dashboards ON DELETE CASCADE
- type TEXT NOT NULL, config JSONB DEFAULT '{}', layout JSONB DEFAULT '{"x":0,"y":0,"w":6,"h":4}'
- created_at, updated_at

**Indexes (after tables):**

- `idx_mindflow_items_board` on items(board_id) WHERE NOT is_archived (partial)
- `idx_mindflow_items_group` on items(group_id)
- `idx_mindflow_items_parent` on items(parent_id) WHERE parent_id IS NOT NULL (partial)
- `idx_mindflow_items_position` on items(board_id, group_id, position)
- `idx_mindflow_items_colvals` on items USING GIN(column_values jsonb_path_ops)
- `idx_mindflow_groups_board` on groups(board_id)
- `idx_mindflow_views_board` on views(board_id)
- `idx_mindflow_widgets_dashboard` on dashboard_widgets(dashboard_id)

**Trigger function + triggers:**

- `mindflow_set_updated_at()` function: sets NEW.updated_at = now(), RETURNS TRIGGER
- BEFORE UPDATE triggers on: boards, groups, items, views, dashboards (5 triggers total)

IMPORTANT: Use `CREATE TABLE IF NOT EXISTS` and `CREATE INDEX IF NOT EXISTS` for idempotency. Use `CREATE OR REPLACE FUNCTION` for the trigger function.

<task type="auto">
  <name>Task 2: Create RPC functions for atomic CRUD operations</name>
  <files>apps/docs/sql/071_mindflow_rpc_functions.sql</files>
  <action>
Create `apps/docs/sql/071_mindflow_rpc_functions.sql` with 5 RPC functions. Follow the exact signatures from RESEARCH.md:

**1. mindflow_update_column_value(p_item_id UUID, p_column_id TEXT, p_value JSONB) RETURNS JSONB:**

- Atomic merge: `SET column_values = column_values || jsonb_build_object(p_column_id, p_value)`
- RETURNING column_values
- SECURITY DEFINER

**2. mindflow_update_column_values(p_item_id UUID, p_values JSONB) RETURNS JSONB:**

- Batch merge: `SET column_values = column_values || p_values`
- RETURNING column_values
- SECURITY DEFINER

**3. mindflow_create_board(p_name TEXT, p_created_by UUID, p_columns JSONB DEFAULT '[]', p_description TEXT DEFAULT NULL) RETURNS JSONB:**

- INSERT board, INSERT default group ('New Group', color '#6C5CE7', position 'a0'), INSERT default view ('Main Table', type 'table', is_default true)
- RETURNS jsonb_build_object('board_id', v_board_id, 'group_id', v_group_id, 'view_id', v_view_id)
- SECURITY DEFINER

**4. mindflow_move_item(p_item_id UUID, p_target_group_id UUID, p_new_position TEXT) RETURNS void:**

- UPDATE items SET group_id, position, updated_at
- SECURITY DEFINER

**5. mindflow_reorder_group(p_group_id UUID, p_new_position TEXT) RETURNS void:**

- UPDATE groups SET position, updated_at
- SECURITY DEFINER

All functions use `CREATE OR REPLACE FUNCTION` and `LANGUAGE plpgsql SECURITY DEFINER`.
</action>

Create `apps/docs/sql/072_mindflow_rls_policies.sql` with:

**Enable RLS on all 6 tables:**

```sql
ALTER TABLE mindflow_boards ENABLE ROW LEVEL SECURITY;
ALTER TABLE mindflow_groups ENABLE ROW LEVEL SECURITY;
ALTER TABLE mindflow_items ENABLE ROW LEVEL SECURITY;
ALTER TABLE mindflow_views ENABLE ROW LEVEL SECURITY;
ALTER TABLE mindflow_dashboards ENABLE ROW LEVEL SECURITY;
ALTER TABLE mindflow_dashboard_widgets ENABLE ROW LEVEL SECURITY;
```

**Policies (Phase 1: owner-based, Phase 7 adds roles):**

- `mindflow_boards_owner` on boards FOR ALL USING (created_by = auth.uid())
- `mindflow_groups_via_board` on groups FOR ALL USING (board_id IN (SELECT id FROM mindflow_boards WHERE created_by = auth.uid()))
- `mindflow_items_via_board` on items FOR ALL USING (board_id IN (SELECT id FROM mindflow_boards WHERE created_by = auth.uid()))
- `mindflow_views_via_board` on views FOR ALL USING (board_id IN (SELECT id FROM mindflow_boards WHERE created_by = auth.uid()))
- `mindflow_dashboards_owner` on dashboards FOR ALL USING (created_by = auth.uid())
- `mindflow_widgets_via_dashboard` on widgets FOR ALL USING (dashboard_id IN (SELECT id FROM mindflow_dashboards WHERE created_by = auth.uid()))

**GRANT ALL on all 6 tables TO authenticated.**

Use `DROP POLICY IF EXISTS ... ON ...;` before each CREATE POLICY for idempotency.
