       - Props: `{ stage: CjmPipelineFlowRow & { clients: CjmClientPosition[] }; onConfigClick: (stageKey: string) => void; onMetricsHover: (stage: CjmPipelineFlowRow, rect: DOMRect) => void; onMetricsLeave: () => void }`
       - Header: stage name + color indicator (use `stage.color` as left border or dot) + client count badge
       - Body: list of `ClientBadge` components for each client in `stage.clients`
       - Footer: owner name (muted text), SLA hours (e.g. "SLA: 24h")
       - Click on header area triggers `onConfigClick(stage.stage_key)`
       - Mouse enter on header triggers `onMetricsHover` with bounding rect for popover positioning
       - Styling: `min-w-[200px] max-w-[260px] flex-shrink-0 bg-bg-primary border border-border-default rounded-lg flex flex-col`
       - Empty state: if no clients, show dashed outline text "Nenhum cliente"

       - Props: `{ pipeline: PipelineMapData; onConfigClick: (stageKey: string) => void; onMetricsHover: ...; onMetricsLeave: ... }`
       - Render stages sorted by `stage_order` as `StageColumn` components in a horizontal flex row with `overflow-x-auto`
       - Add connecting arrows between stages (simple CSS `>` chevron or a 1px line with `after:` pseudo-element — keep it minimal)

    5. **`JourneyCanvas.tsx`** — Top-level swimlane container:
       - Props: `{ pipelines: PipelineMapData[]; onConfigClick: (stageKey: string) => void }`
       - State: `selectedPipeline` index (default 0), `metricsPopover: { stage: CjmPipelineFlowRow; rect: DOMRect } | null`
       - Render pipeline tab buttons at top (same styling as page tabs but smaller — pill buttons)
       - Render `PipelineLane` for the selected pipeline
       - Render `StageMetricsPopover` positioned absolutely near `metricsPopover.rect` when not null
       - Handle `onMetricsHover` / `onMetricsLeave` to show/hide popover (use 200ms delay on leave to avoid flicker)

    6. **Update `CustomerJourney/index.tsx`**:
