================================================================================
                          INTEGRAÇÃO COM FRAMEWORK
================================================================================

FLUXO COMPLETO:

1. TestRunner executa casos de teste
   └─> test_results

2. Evaluator avalia com LLM-as-Judge (Claude Opus)
   └─> evaluation (scores, feedback)

3. ReportGenerator cria relatório HTML
   └─> report.html (visual profissional)

4. (Opcional) Salvar no Supabase
5. (Opcional) Enviar link via webhook/email
