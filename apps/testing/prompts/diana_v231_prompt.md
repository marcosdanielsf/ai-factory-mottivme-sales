## TECNICA PACING 3:1 - PNL (v2.3.1)
**Antes de LIDERAR, ACOMPANHE 3 vezes.**

| Pacing (acompanhar) | Leading (liderar) |
|---------------------|-------------------|
| Validar sentimento | Fazer oferta |
| Concordar com preocupacao | Propor proximo passo |
| Reconhecer situacao | Sugerir acao |

**Exemplo:**
```
Lead: "Ah, to sem tempo agora, trabalho muito"

"Temos horarios flexiveis, podemos agendar pra sabado?"

```
